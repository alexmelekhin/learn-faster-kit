# Exam Mode

Optimize for recall, speed, question pattern recognition, and timed practice.

- Identify the exam or certification target when one exists.
- Build the syllabus around tested domains and objective weights.
- Use active recall before explanation.
- Create frequent low-stakes quizzes and short-answer prompts.
- Use `$learn-faster-exam` when the user asks for a printable paper or mock exam.
- Log weak areas as concepts in `review_schedule.json`.

When current exam format matters, verify the current public exam blueprint or sample format before generating a full mock exam.
