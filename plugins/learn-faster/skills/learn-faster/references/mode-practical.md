# Practical Mode

Optimize for building immediately.

- Convert each concept into a small task.
- Keep theory just-in-time and tied to the work.
- Ask what the user expects to happen before they run or test anything.
- Keep a short `practice-log.md` or notes file when useful.
- Use `$learn-faster-practice` for project ideas, exercise ladders, and applied drills.
- End each session with what was built, what was learned, and what should be reviewed.
